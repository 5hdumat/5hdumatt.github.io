HTTP 메서드

HTTP 주요 메서드 종류 HTTP 메서드는 클라이언트가 서버에 요청을 할 때 서버로부터 기대하는 행동이다. 주로 많이 사용하는 HTTP 메서드는 아래와 같다. GET GET은 이름 그대로 리소스를 조회하기 위한 메서드이다. 서버에 전달하고 싶은 데이터는 query(쿼리 파라미터, 쿼리 스트링)를 통해서 전달한다. GET 메서드도 메시지 바디를 통해 데이터를 전달할 수 있지만, 지원하지 않는 서버들이 많아 권장하지 않는다. 리소스 조회 과정을 살펴보자. 리소스 조회 예제 GET 메서드와 함께 /members/100 경로(path)의 리소스를 서버에 요청한다....
