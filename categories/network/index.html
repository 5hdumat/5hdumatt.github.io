URI

URI(Uniform Resource Identifier) URI는 리소스를 식별하는 통합된 방법이라는 의미이다. 사람을 식별할 때는 주민등록번호를 이용하는 것 처럼 리소스가 어디있는지 식별하는 방법을 URI라고 한다. Uniform: 리소스 식별하는 통일된 방식 Resource: 리소스, URI로 식별할 수 있는 모든 자원 (제한 X) 즉, URI로 식별할 수 있는 모든 정보를 리소스라고 한다. Identifier: 다른 항복과 구분하는 데 필요한 정보 예) 사람을 식별할 때 사용하는 주민등록번호가 사람의 Identifier가 된다. “URI는 로케이터(locator), 이름(name) 또는 둘다 추가로 분류될 수 있다....

DNS

DNS? DNS는 무엇이고 왜 사용하는 걸까? 그 이유는 위 예제처럼 IP는 기억하기 어렵다는 점이고 또 다른 이유는 IP는 언제든지 변경될 수 있다는 점이다. 예를 들어 200.200.200.2로 기억해두었던 IP가 200.200.200.3으로 변경되었을 때 기존 IP는 당연히 접근 불가능하게 된다. 이를 해결하려면 도메인 네임 시스템(DNS, Domain Name System) 을 이용해야 한다. DNS의 특징 DNS는 등록한 도메인 명을 IP 주소로 변환해 준다. 쉽게말해 중간에서 전화번호부 서버같은 서비스를 제공해 준다고 이해하면 된다. 기철이한테 전화를 걸면 기철이의 핸드폰번호로 전화가 걸리는 것 처럼 말이다....
