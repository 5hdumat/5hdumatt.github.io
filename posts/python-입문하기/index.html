Python 입문하기

May 16, 2021 · 서민규

언어는 Python으로 하겠습니다. 근데 이제 알고리즘을 곁들인#

C/C++이 아닌 Python을 선택한 이유#

초보자도 쉽게 접근할 수 있는 언어, 수많은 자료구조 모듈, 짧은 코드길이, 쉬운 문자열 핸들링, 신경 쓰지 않아도 되는 오버플로우, 인터프리터 방식의 빠른 피드백은 Python의 대표적인 장점들입니다.

또한 대다수의 기업들이 Coding Interview에서 Python을 지원하고 있어 알고리즘 공부를 시작한 저에겐 이 언어를 선택 해야 할 이유가 너무나도 명확하고 심플했습니다.

Python 시작하기 (Mac OS)#

Python을 설치하기에 앞서 Mac OS 패키지 관리 도구인 homebrew부터 설치하기로 했습니다.

homebrew 설치

하단의 Command를 복사하여 터미널에 붙여넣기 합니다.

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)

Python 설치

1. Python Version Manager 설치#

언어는 기본적으로 주기적인 업데이트가 이루어집니다. 업데이트가 이루어질 때마다 새로운 버전의 개발 환경을 재설치하는 것은 매우 비효율적이고 번거로운 일이 아닐 수 없습니다.
하여 Python의 인터프리터 환경을 손쉽게 변경시켜주는 Version Manager를 설치하도록 합니다.

brew update

brew install pyenv

2. pyenv를 이용한 Python 설치#

  • 설치 가능한 인터프리터 리스트를 확인합니다.
pyenv install --list

#

  • 필자의 경우 가장 최신 버전인 anaconda3-5.3.1를 설치했습니다.
pyenv install anaconda3-5.3.1

#

  • 정상적으로 설치되었는지 확인합니다.
pyenv version

$ anaconda3-5.3.1 (set by /Users/home/.pyenv/version)가 출력되었습니다.

#

  • Python 버전을 확인합니다.
Python -V

Python 2.7이 기본 설치되어 있는 걸 확인하실 수 있습니다.

#

  • 파이썬 버전을 anaconda3-5-3-1로 변경합니다.
pyenv global anaconda3-5.0.1

명령어를 실행했음에도 Python 버전이 변경되지 않을 경우 하단 명령어를 실행시켜줍니다.

eval "$(pyenv init -)"

#

  • Python 버전을 다시 확인합니다.
Python -V

Python 3.7로 버전이 변경되었습니다.

#

  • 파이썬 설치가 완료되었습니다.
