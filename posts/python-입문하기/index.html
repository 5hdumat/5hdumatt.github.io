Python 입문하기

May 16, 2021 · 서민규

언어는 Python으로 하겠습니다. 근데 이제 알고리즘을 곁들인#

Python을 선택한 이유#

초보자도 쉽게 접근할 수 있는 언어, 수많은 자료구조 모듈, 짧은 코드길이, 쉬운 문자열 핸들링, 신경 쓰지 않아도 되는 오버플로우는 Python의 대표적인 장점들 입니다.

또한 대다수의 기업들이 Coding Interview에서 Python을 지원하고 있어 (알고리즘 공부를 시작한 저에겐) 이 언어를 선택 해야 할 이유가 너무나도 명확하고 심플했습니다.

Python 시작하기 (Only Mac)#

Python을 설치하기에 앞서 Mac OS 패키지 관리 도구인 homebrew부터 설치합니다.

homebrew 설치

하단의 Command를 복사하여 터미널에 붙여넣기 합니다.

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)

#

pyenv와 pyenv-virtualenv를 이용한 Python 개발 환경 구성

인터프리터 환경은 주기적인 업데이트가 이루어집니다.
그럴때마다 새로운 버전의 개발 환경을 재설치하는 것은 매우 귀찮고 번거로울 것 입니다.
Python을 버전별로 관리 할 수 있도록 Python Version Manager인 pyenv를 설치합니다.

brew update
brew install pyenv

#

이어서 Python 프로젝트를 가상환경을 이용해 분리 운영 할 수 있도록 pyenv-virtualenv를 설치합니다.
가상환경은 프로젝트 각각에 설치된 패키지들 사이의 충돌을 막아 주기 위해 필요합니다.

brew install pyenv-virtualenv

#

pyenv및 pyenv-virtualenv가 정상동작 할 수 있도록 셀 설정파일에 각 프로그램의 초기화 코드를 추가하여 환경변수를 설정합니다.
현재 제가 사용하고 있는 Mac OS 버전은 Big Sur입니다.
Catalina 이후 버전부터 기본 셸이 zsh로 변경되었기 때문에 ~/.zshrc 파일을 수정했습니다.

vi ~/.zshrc

export PATH=$HOME/.pyenv/bin:$PATH
eval "$(pyenv init -)"
eval "$(pyenv virtualenv-init -)"

#

설정을 저장합니다.

source ~/.zshrc   #설정 저장 - zsh

exec $SHELL   #shell 재실행

#

버전에 맞게 Python을 설치합니다. 필자는 3.7.7 버전을 설치하기로 했습니다.

pyenv install 3.7.7

#

Mac OS 버전이 Big sur로 버전업 되면서 XCode 관련 업데이트가 있었는지 BUILD FAILED 에러가 발생했습니다.
급한대로 하단 명령어를 실행하여 Python을 설치했습니다.

CFLAGS="-I$(brew --prefix openssl)/include -I$(brew --prefix bzip2)/include -I$(brew --prefix readline)/include -I$(xcrun --show-sdk-path)/usr/include" LDFLAGS="-L$(brew --prefix openssl)/lib -L$(brew --prefix readline)/lib -L$(brew --prefix zlib)/lib -L$(brew --prefix bzip2)/lib" pyenv install --patch 3.7.7 < <(curl -sSL https://github.com/python/cpython/commit/8ea6353.patch\?full_index\=1)

#

가상환경 명령어를 상황에 맞게 실행합니다.

python -V # 2.7.7 출력
pyenv virtualenv 3.7.7 workspace # 가상환경 생성 pyenv virtualenv <version> <virtualenv_name> 
pyenv activate workspace
python -V # 3.7.7 출력
pyenv deactivate # 가상환경 해제하기
